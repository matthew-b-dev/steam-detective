# Steam Detective

A daily Steam game guessing puzzle. Uncover clues to identify the mystery game!

## Features

- **Daily Puzzles**: New game to guess every day
- **Two Difficulty Levels**: Easy and Expert case files
- **Progressive Clues**: Reveal clues strategically to identify the game
- **Score Tracking**: Compare your score with other players
- **Ad-Free**: No ads, no tracking, just puzzles

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Supabase
