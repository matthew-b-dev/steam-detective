# Steam Detective

A daily Steam game trivia puzzle. Uncover clues to identify the mystery game!

Play the live version at https://steamdetective.wtf

## Features

- **Daily Puzzles**: New games to guess every day
- **Progressive Clues**: Reveal clues strategically to identify the game
- **Score Tracking**: Compare your score with other players globally
- **Ad-Free**: No ads or monetization

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Tech Stack

- React 19
- TypeScript
- Vite
- Framer Motion
- PostgreSQL/Supabase
